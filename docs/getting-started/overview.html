Argon - Design System

An user-friendly and beautiful design system based on Bootstrap 4 and inspired by Stripe.

We at Creative Tim have always wanted to deliver great tools to all the web developers who are using our products because we want to see better websites and web apps on the internet.

Why Stripe as inspiration?

It’s pretty simple, Stripe.com it’s great and they created a design standard that a lot of designers and companies get inspiration from. Please search “Gradient” on Dribbble and you will see a lot of similarities.

Developer First

Since Stripe is a "Developer First" company we decided that they are a great inspiration for our next release.

High quality before everything

They put a lot of effort in building great interfaces and pay a lot of attention to details, more details in their blogpost: " Connect: behind the front-end experience"

Community helpers

They also give resources to developers, like these fully coded components on GitHub.

So we decided to check their resources and come with this Design System that is fully coded on top of Bootstrap 4 in order to help the developers community create better websites.
